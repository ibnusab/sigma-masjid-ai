# 📖 About

**Sigma Masjid AI** adalah aplikasi Android modern yang dirancang untuk membantu pengurus masjid mengelola seluruh aktivitas operasional secara digital.

Mulai dari administrasi jamaah, keuangan, zakat, inventaris, agenda, hingga pembuatan laporan berbasis **Google Gemini AI** dalam satu aplikasi yang cepat, aman, dan mudah digunakan.

# 🤖 Google Gemini AI

Sigma Masjid AI menggunakan **Google AI Studio (Gemini API)** untuk membantu pengurus masjid.

### AI Assistant

Contoh:

* Ringkas laporan keuangan
* Membuat pengumuman
* Membuat surat resmi
* Menjawab pertanyaan jamaah
* Membuat jadwal imam
* Analisis pemasukan
* Prediksi pengeluaran
* Membuat notulen rapat
* Menyusun proposal kegiatan
* Rekomendasi anggaran kegiatan
